## Description
A C-based Implementation of the Vigenère Cipher.

## Installation
* **Compile and Execute on GNU/Linux using GCC**
```
$ gcc vigenere.c -o vigenere
$ chmod +x vigenere
$ ./vigenere
```

* **Compile and Execute on Windows NT using the VS Developer Command Prompt**
```
$ cl vigenere.c
$ .\vigenere.exe
```
